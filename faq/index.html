FAQ
How do I apply it?
All you have to do us navigate to the substratum application and enable+ build the overlays after tapping on the PitchBlack tile. If you already have a theme applied, head over to the overlay manager and disable all overlays before you enable+build PitchBlack than reboot your device
What ROMs does the theme support?
PitchBlack being a theme intended to be utilized via the substratum application automatically supports all the ROMs substratum does. You can easily find a list of supported ROMs by heading over to this link
What if the theme updates?
Just build+enable again to make sure you're rocking the best version of the theme
Does PitchBlack supports theme ready gapps?
Officially it does not support ! Not recommended
Help!!! Unexpected bootloop!
Chill, all you've got to do is use the rescue option of substratum
